7 February 2024

I have always been interested in the software that people use. In fact, you could say that I’m so interested that it is a great source of procrastination - focusing on the tools as opposed to the content.

This is another insight I found first on Manu’s blog. It is quite a trend. At the time of writing 354 people have shared their defaults. The full list is curated by Robb Knight.

Here are my defaults as of February 2024. They only include personal applications and side projects. At work it is an exclusively Microsoft affair.

  • Mail Client: Apple Mail / Fastmail App
  • Mail Server: Fastmail
  • Notes: Ulysses / Drafts / Apple Notes
  • To-Do: Apple Notes / Reminders
  • Photo Shooting: Apple Camera
  • Photo Management: Apple Photos
  • Calendar: Apple Calendar
  • Cloud File Storage: iCloud
  • RSS: Reeder (with iCloud)
  • Contacts: Apple Contacts, synced via Fastmail.
  • Browser: Safari, DuckDuckGo, Brave (iOS)
  • Chat: Apple Messages, Telegram, WhatsApp (not my preference)
  • Mastodon Client: IceCubes
  • Bookmarks: Safari
  • Read It Later: Safari / Reeder / Open tabs (no consistency)
  • Word Processing: Ulysses / Pages
  • Spreadsheets: Numbers
  • Presentations: PowerPoint (purely at work).
  • Shopping Lists: Shared Apple Notes grocery list
  • Meal Planning: An ad hoc, weekly scramble.
  • Budgeting and Personal Finance: Various banking apps.
  • News: Naked Capitalism / Financial Times / The Guardian / RSS Feed
  • Music: Apple Music
  • Podcasts: Pocket Casts
  • Password Management: Bitwarden
  • Search: DuckDuckGo
  • Code Editor: BBEdit
  • Books: Amazon Kindle / Calibre
  • PDFs: PDF Expert
  • Reference Management: DevonThink
  • Weather: BOM Weather
  • Ad Blocking: 1Blocker
