I ran across a post somewhere referencing this and it seems that a bunch of people are doing lists of their 'defaults'.

Seems like fun.

  • Mail Client: mutt
  • Mail Server: postfix
  • Notes: A Pilot Metropolitan medium nib and whatever paper is nearby.
  • To-Do: Same as notes.
  • Photo Management: Some POSIX shell scripts.
  • Calendar: Calendar.app via CalDAV
  • Cloud File Storage: No thanks.
  • RSS: Reeder and NetNewsWire via FreshRSS
  • Contacts: Contacts.app via CardDAV
  • Browser: Safari
  • Chat: irssi, SMS.
  • Bookmarks: Bespoke Machinery.
  • Word Processing: vi(1), Word for work.
  • Spreadsheets: Numbers, Excel for work.
  • Presentations: Powerpoint
  • Shopping Lists: A Pilot Explorer fine nib and crappy notebook.
  • Meal Planning: See Notes.
  • News: RSS
  • Music: NFS into whatever Apple re-branded iTunes, foobar2000.
  • Podcasts: Bespoke machinery feeding PocketCasts.
  • Password Management: pass(1)
  • Code editing: vi(1)
  • Books: Actual books.
  • Audiobooks: BookPlayer
