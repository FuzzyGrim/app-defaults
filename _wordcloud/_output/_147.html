  • To-Do : Apple Rappels, Notion
  • Photo Management : Apple Photos, Nextcloud (auto-hébergé)
  • Calendar : Google Agenda
  • Cloud File Storage : Nextcloud (auto-hébergé), Google Drive
  • RSS : readerself (auto-hébergé)
  • Contacts : Google Contacts
  • Browser : Mozilla Firefox, Google Chrome
  • Chat : Apple Messages, Signal, Whatsapp
  • Bookmarks : Shaarli (auto-hébergé)
  • Read It Later : Shaarli (auto-hébergé)
  • Word Processing : LaTeX, Microsoft Word, Google Docs
  • Spreadsheets : Microsoft Excel, Google Sheets
  • Presentations : Microsoft PowerPoint, Google Slides
  • Shopping Lists : Apple Rappels
  • Meal Planning : n/a
  • Budgeting and Personal Finance : un tableur personnalisé
  • News : Brief.me, Brief.eco
  • Music : Spotify
