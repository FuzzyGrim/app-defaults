I want to get some inspiration and find via Jens Comiotto-Mayer’s blog that there is a Duel of the Defaults! which asks the question about our standard apps. What a great idea! I love spotting what apps other people use. I always come across great new apps that way. I’ll join in right now!.

Because this is international and Robb keeps an overview list, here again and below the list.

  • Mail Client: Apple Mail and mailbox.org webclient
  • Mail Provider: mailbox.org
  • Notes: Obsidian
  • iPhone Photo Shooting: iOS Camera App
  • Photo Management: Adobe Lightroom Classic
  • Calendar: Apple Calender with self hosted ical
  • Cloud Storage: self hosted Nextcloud
  • Contacts: Apple Contacts with self hosted carddav
  • Browser: Chrome Beta
  • Search Engine: Google
  • Music: Spotify
  • Password Management: Self hosted Bitwarden
  • Terminals: iTerm2 & Termius
  • Code Editor: Visual Studio Code
  • Code Management: self hosted Gitea
  • Social Media: Mastodon with IceCubes
