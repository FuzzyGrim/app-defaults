Default Apps

I love a good blogging trend. So much so that I restarted this blog (I also missed micro-blogging, but that's another story).

  • Mail Client: Fastmail in the browser
  • Notes: Bear Notes (though mostly on paper)
  • To-Do: Apple Reminders
  • Photo Shooting: Apple Camera and Moment Camera (I don't use my phone for pictures much)
  • Photo Management: Apple Photos and Lightroom
  • Calendar: Fantastical with Google Calendar, Microsoft Outlook, iCloud Calendar
  • Cloud File Storage: Apple iCloud
  • RSS: Reeder with Feedbin
  • Contacts: Apple Contacts
  • Browser: Arc on desktop, Safari on mobile
  • Chat: Discord, Telegram, Whatsapp, iMessage
  • Read It Later: Switching between Matter and Readwise
  • Word Processing: iA Writer
  • Spreadsheets: Microsoft Excel
  • Presentations: I seldom do presentations, but Microsoft PowerPoint
  • Shopping Lists: Bring
  • Meal Planning: I don't plan meals, but I track my food via MacroFactor
  • Budgeting and Personal Finance: I budget straight in my banking app
  • News: I don't read news
  • Music: Spotify
  • Podcasts: Overcast
  • Password Management: 1Password
  • Code Editor: I write most of my code in Jupyter Notebooks

I discovered this first via Matt Birchler.
