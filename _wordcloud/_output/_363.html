Inspired by Robb and the Hemispheric Views podcast, here’s a list of some of the apps I use for some of the things. For other things I use, see my Uses page. So many other people have done this, and I find it fascinating to see the range of tools and approaches.

  • 📨 Mail Client: Gmail
  • 📮 Mail Server: Gmail
  • 📝 Notes: Obsidian
  • ✅ To-Do: Todoist (personal), Flow (household) 1, GitHub issues (work)
  • 📷 Photo Shooting: stock camera app on my Pixel 3A
  • 📆 Calendar: Google Calendar
  • 📁 Cloud File Storage: Google Drive for things that need sharing outside my household, Paperless NGX otherwise
  • 📖 RSS: Inoreader
  • 🌐 Browser: Firefox (yes, on mobile too)
  • 💬 Chat: Telegram, Discord, WhatsApp
  • 🔖 Bookmarks: Firefox’s built-in bookmarks (rarely)
  • 📑 Read It Later: Inoreader
  • 📜 Word Processing: Obsidian
  • 📈 Spreadsheets: VSCode
  • 📊 Presentations: Nope (thankfully!)
  • 🛒 Shopping Lists: Flow
  • 💰 Budgeting and Personal Finance: my bank’s app
  • 🎵 Music: my own Plex server
  • 🎤 Podcasts: Pocket Casts
  • 🔐 Password Management: Bitwarden

And a few bonus ones:

  • 🐘 Social Media: social.lol via Elk
  • 🚀 Launcher: Raycast
  • 📘 Code: VSCode, PyCharm
  • 🖥️ Terminal: Wezterm, prompt from Starship and autocomplete from fig

Footnotes

  1. Why yes, my partner and I did write our own sizeable chunk of software to remind us to do the laundry, feed the cat and buy onions. You’re welcome to use it too (let me know if you do!), but we make no promises that it actually works. ↩
