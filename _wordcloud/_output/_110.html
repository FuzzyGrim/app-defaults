2023年11月11日

看到 Dayu 的分享和 App Defaults 的项目，也分享一下自己使用的工具。

📨 Mail Client

  • Spark 💰 最喜欢多端同步功能

📮 Mail Server

  • iCloud 💰 iCloud+ 的域名邮箱便宜够用
  • CloudFlare 只来收件，如果想做真正的邮箱，参考 Lepture 的分享, 发件推荐使用 Resend

📝 Notes

  • Apple Notes 随手记
  • Obsidian 工作记录，公司安全要求，不可以使用同步功能

✅ To-Do

  • Apple Reminders

📷 iPhone Photo Shooting

  • Apple Camera

🖼️ Photo Management

  • iCloud 💰

📆 Calendar

  • Apple Calendar

📁 Cloud File Storage

  • iCloud 💰 照片备份，文件备份
  • openmediavault 影视存储，PT 下载

📖 RSS

  • Inoreader 免费版 150 订阅足够，保持 100 上下高质量订阅源，够看也不浪费时间
  • NetNewsWire macOS 上免费且漂亮的 RSS 客户端，配合 Inoreader 使用

🙍🏻‍♂️ Contacts

  • Apple Contacts

🌐 Browser

  • Edge 多端历史记录同步，推荐 Better History 插件，更漂亮。

💬 Chat

  • Wechat 脱离不了的“小而美”
  • Telegram

🔖 Bookmarks

  • Browser

📰 News

  • Twitter

  • Mastodon

🎵 Music

  • Apple Music 💰 听一些 QQ 音乐没有的

  • QQ Music 💰 中文最全

🎤 Podcasts

  • Google Podcasts 可惜要关闭了，寻找替代品中

🔐 Password Management

  • Enpass 💰 8年前，$30 买的终身版，应该还能再使用8年。
