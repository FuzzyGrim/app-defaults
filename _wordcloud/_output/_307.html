First I saw it on Robb Knight’s website and said to myself “yeah look, another one with a list of his tools”. Then I saw it on Garrit’s blog, then on Chris Coyier’s and today on Cory Dransfeld’s and Chris McLeod’s and I thought: funny trend with the App Defaults. Let’s go for it…

  • Mail:   Gmail
  • Tasks:   Google Tasks via Business Calendar 2, Trello
  • Contacts:   Google Contacts
  • Photo Shooting:   Nikon D500, Google Pixel 8
  • Photo Editing/Management:   Adobe Lightroom Classic
  • RSS:   Feedly
  • News:    Google News, Feedly
  • Browser:   Google Chrome, Microsoft Edge, Vivaldi
  • Chat:   Signal
  • Bookmarks:   Google Chrome, kiko.io’s Tiny Tools
  • Read It Later:   Trello, Google Chrome (Android)
  • Word Processing:   M365
  • Spreadsheets:   M365
  • Presentations:   M365
  • Music:   BlackPlayer
  • Passwords:   KeePass
  • Code Editor   Visual Studio Code, Visual Studio
  • Source Control   GitHub, OneDrive
  • Shopping Lists:   Google Keep
  • Meal Planning:   Leni - Ultimate version of wife (no link available)

I used to be called the “tool boy” because I had to try out every damn tool and accordingly have many on every type of device. But this list is the stuff I deal with every day.
