I'm seeing so many create their own list of default apps with Rob Knight's encouragement. Robb is the one who started this thing. For your consideratio, I hereby offer my defaults (in no particular order).

  • calendar: Google Calendar in Chrome (tab 1)
  • mail: Google Mail in Chrome (tab 2)
  • browser: you guessed it, Chrome
  • password manager: 1password
  • code editor: VS Code
  • window manager: magnet
  • os: macOS Sonoma
  • RSS reader: Feedly in Chrome (tab 4)
  • site analytics: Fathom in Chrome (tab 3)
  • all around utility: Alfred
  • mastodon client: Ivory
  • notes: Obsidian
  • todo: Obsidian
  • spreadsheet: Google Sheets
  • documents: Google Docs
  • music: YouTube Music
  • photo management: Google Photos
  • grocery shopping list: Alexa
  • cloud storage: Dropbox
  • cloud backup: Backblaze
  • podcasts: Overcast
  • contacts: Apple Contacts
  • personal finance: Google Sheets
  • maps: Google Maps
  • static site generator: Eleventy

In case it's not apparent, like Cory Dransfeldt, for the most part, I'm pretty much all in on Google.
